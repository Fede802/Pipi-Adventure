# Pipi-Adventure
Developed with java15
Tested on Windows 10,11

To run the application, jar must be in the same folder of config and res

Added some variables for debug purpose: 
    
    in game screen:
    -wallCollision = true           
    -enemiesCollision = true
    -infiniteBullets = false

    in upgrade screen  
    -freeUpgrade = false

key to change the initial state:
    -W = wallCollision
    -E = enemiesCollision
    -B = infiniteBullets
    -C = freeUpgrade

if you change the state:
    -in game screen the corresponding letters are shown in the bar
    -in the upgrade screen the price shown is 0
